## 17. Project 2

Project Title:
RAG-Based AI Chatbot for Document Question Answering

Description:
Developed a Retrieval-Augmented Generation chatbot that answers user questions from uploaded PDF documents using LLM and vector database.

Technologies Used:
Python, LangChain, OpenAI API, ChromaDB, FastAPI, Streamlit

Responsibilities:
- Loaded and processed PDF documents.
- Split documents into chunks.
- Created embeddings and stored them in vector database.
- Retrieved relevant context based on user query.
- Generated accurate answers using LLM.
- Built frontend using Streamlit.

## 18. Project 3

Project Title:
Customer Churn Prediction using Machine Learning

- Handled missing values and categorical encoding.
- Applied Logistic Regression, Random Forest, and XGBoost.
- Evaluated model using accuracy, precision, recall, and F1-score.
- Created visual reports for business understanding.
